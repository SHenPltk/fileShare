# 📁 局域网文件共享 (fileShare)

这是一个基于 Python Flask 开发的轻量级局域网文件共享工具。它允许你在同一局域网内的不同设备之间轻松地上传、下载和管理文件。

## ✨ 功能特性

- **🚀 拖拽上传**：支持将文件直接拖入浏览器窗口进行上传。
- **📊 进度显示**：实时显示文件上传进度条。
- **📥 文件下载**：一键下载共享目录中的所有文件。
- **🗑️ 文件管理**：支持在线删除不需要的文件。
- **📱 跨设备支持**：自动获取本机 IP，方便手机或其他电脑访问。
- **🎨 现代 UI**：简洁直观的用户界面，良好的交互反馈。

## 🛠️ 快速开始

### 1. 克隆仓库
```bash
git clone https://github.com/SHenPltk/fileShare.git
cd fileShare
```

### 2. 安装依赖
确保你已安装 Python 3，然后安装必要的库：
```bash
pip install flask flask-cors
```

### 3. 运行程序
```bash
python fileshare.py
```

### 4. 访问服务
程序启动后会显示类似以下的信息：
```
=== 文件共享服务已启动 ===
在浏览器访问（其他设备）: http://192.168.1.100:5000
存储目录: /your/path/fileShare/shared_files
```
在同一局域网下的任何设备浏览器中输入显示的地址即可开始使用。

## 📂 项目结构

- `fileshare.py`: 后端逻辑处理。
- `templates/index.html`: 前端交互界面。
- `shared_files/`: 默认的文件存储目录（自动创建）。
- `.gitignore`: Git 忽略配置。

## 📝 注意事项

- 请确保你的防火墙允许 5000 端口的入站连接。
- 建议在受信任的局域网环境中使用。
